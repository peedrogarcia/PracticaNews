# 📰 Práctica - News App

Aplicación Android desarrollada en Kotlin cuyo objetivo principal es consumir y mostrar noticias desde una API pública.

## 🚀 Descripción

Este proyecto nace como una práctica enfocada en la integración de APIs dentro de una aplicación móvil. La app realiza peticiones a un servicio externo de noticias y muestra los resultados de forma sencilla en la interfaz.

## 🛠️ Tecnologías utilizadas

* Kotlin
* Android Studio
* API REST (consumo de datos en tiempo real)
* JSON parsing

## 📱 Funcionalidades

* Conexión a una API de noticias
* Obtención de datos en tiempo real
* Visualización básica de artículos
* Manejo de peticiones HTTP

## 🎯 Objetivo del proyecto

El propósito principal de esta aplicación es demostrar:

* Integración con APIs externas
* Manejo de datos remotos
* Estructura básica de una app Android

## 📌 Estado

Proyecto de práctica / aprendizaje.

## 🤝 Notas

Este proyecto está enfocado en la parte técnica (conexión y consumo de API), por lo que la interfaz es sencilla y funcional.
